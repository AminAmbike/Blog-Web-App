# Blog-Web-App

Blog website App

# Languages and Technologies:
**Front-End**
- JavaScript (EJS)
- HTML
- CSS

**Back-End**
- MongoDB (mongoose)
- Node.js

# Instructions
1. CD into directory containing files using the command terminal
2. run "npm i"
3. run "npm init"
4. run "node app.js"
5. In your browser go to "localhost:3000/"
